Assignment 1: Blink!


Overview

For this assignment I created a schematic for three LEDs that are connected to my arduino. Then I implemented the circuit on a breadboard to create three independent blinking lights.

Schematic



To understand how I was going to get my LEDs to independently light up I sketched a schematic. Because I wanted them to blink independently, I started each LED from a different pin (13, 12, 11) that's on the Arduino board. First, I knew that I wanted to run my LEDs at around 20 mA current. Using Ohm's Law (V = I * R) I calculated the resistances that I needed for my circuit.

I found two different resistances. One needed for both the Green/Red LEDs and one for the White LED, because the White LED has a different voltage drop than the Green and Red LEDs.

I started with Ohm's law: V = I * R

To find resistance (R) based on our two givens, voltage (V) and current (I), I rearranged the equation to be:
R = V/ I

For the Red and Green LEDs:
The voltage source, my computer, was 5 V
The voltage drop for Red and Green is 1.8 V
I want the current to be 20 mA, which is 0.02 A
And I want to find the needed resistance, R Ω

Plugging in those values to the equation I get:
5V - 1.8V = 0.02A * RΩ

Then I solved for R (Ω)
3.2V / 0.02A = RΩ
160Ω = R

For the Red/Green LEDs, I found that to have a current of 20mA and a voltage of 3.2V I needed 160Ω for resistance. The closest resistor in my kit was the 220Ω resistor; this is what I used for my circuit.
With the 220Ω and voltage of 3.2V
I = V / R
I = 3.2V / 220Ω
I = 0.0145A
It's okay that the current is slightly lower than the 0.02A preffered.

For the White LED:
The voltage source, my computer, was 5 V
The voltage drop for White is 3.3 V
I want the current to be 20 mA, which is 0.02 A
And I want to find the needed resistance, RΩ

Plugging in those values to the equation I get:
5V - 3.3V = 0.02A * RΩ

Then I solved for R (Ω)
1.7V / 0.02A = RΩ
85Ω = R

For the White LED, I found that to have a current of 20mA and a voltage of 1.7V I needed 85Ω for resistance. The closest resistor in my kit was the 100Ω resistor; this is what I used for my circuit.
With the 100Ω and voltage of 1.7V
I = V / R
I = 1.7V / 100Ω
I = 0.017A
It's okay that the current is slightly lower than the 0.02A preffered.

Arduino Code

To get the lights to blink independently I coded the Arduino. I found inspiration from the example code snippet, "Blink."

For the setup() I initated the three different pins on the Arduino (13, 12, 11).

The loop() is where the LED lights actually light up and blink. Because I set each LED light to a different pin on the Arduino board, I had control over each light. I decided to make the green light blink twice in a row quickly, and then have the white light blink once, and then have the red light blink once; then repeat until "forever. 


              // Lia Johansen
              // Assignment 1: Blink
              // January 9, 2020

              // the setup function runs once when you press reset or power the board
              void setup() {
                // initialize digital pin 13 (this is the green LED)
                pinMode(13, OUTPUT);
                // initialize digital pin 12 (this is the white LED)
                pinMode(12, OUTPUT);
                // initialize digital pin 11 (this is the red LED)
                pinMode(11, OUTPUT);
              }


              // the loop function runs over and over again forever blinking the LED lights
              void loop() {
                // turn the green LED on by making the voltage HIGH
                digitalWrite(13, HIGH);
                // wait for half of a second while green LED is on
                delay(500);
                // turn the green LED off by making the voltage LOW
                digitalWrite(13, LOW);
                // wait for 1/20 a second
                delay(50);

                // turn the green LED on again by making the voltage HIGH
                digitalWrite(13, HIGH);
                // wait for half of a second while green LED is on
                delay(300);
                // turn the green LED off by making the voltage LOW
                digitalWrite(13, LOW);
                // wait for 1/10 a second
                delay(100);


                // turn the white LED on by making the voltage HIGH
                digitalWrite(12, HIGH);
                // wait for half a second while white LED is on
                delay(500);
                // turn the white LED off by making the voltage LOW
                digitalWrite(12, LOW);
                // wait for 1/10 of a second
                delay(100);


                // turn the red LED on by making the voltage HIGH
                digitalWrite(11, HIGH);
                // wait for half a second while red LED is on
                delay(500);
                // turn the red LED off by making the voltage LOW
                digitalWrite(11, LOW);
                // wait for 1/10 of a second
                delay(100);
              }
